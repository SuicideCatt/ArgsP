# ArgsP
C++ Arguments parser

### Build
Dependents:
- Git
- CMake
- C++ 20

Step 1: Download
```bash
git clone URL
cd ArgsP
mkdir build
cd build
```

Step 2: Linux
```bash
$ cmake ..
$ make -j$(nproc)
# make install
```
OR Step 2: Linux no compile
```bash
$ cmake .. -DSCT_NO_BUILD
# make install
```
